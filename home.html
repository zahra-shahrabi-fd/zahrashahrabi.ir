What is called a web designe process exactly?

Web design is the process of planning, conceptualizing, and arranging content intended for the Internet. Modern web design goes beyond how things look (aesthetics) to include how things work (functionality). Web design is not limited to websites as it includes other uses such as web apps, mobile apps, and user interface design.
in today’s mobile first world, there are only two ways to properly design a website: adaptive and responsive websites. Adaptive web design uses two or more versions of a website that are customized for different screen sizes. Responsive websites use a combination of flexible grids (based on percentages) with breakpoints (using media queries) to create a custom look at every screen size. Unlike adaptive sites that adapt only when they hit a breakpoint, responsive websites are constantly changing according to the screen size.
Todays, the most common framework that used in the website's responsive process is called Bootstrap.
Bootstrap is the most popular CSS Framework for developing responsive and mobile-first websites.
